Приведите пример структуры HTML5 документа.
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="Описание страницы">
    <title>Заголовок страницы</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <header>
        <h1>Логотип сайта</h1>
        <nav>
            <ul>
                <li><a href="#home">Главная</a></li>
                <li><a href="#about">О нас</a></li>
                <li><a href="#contact">Контакты</a></li>
            </ul>
        </nav>
    </header>

    <main>
        <article>
            <h2>Заголовок статьи</h2>
            <p>Содержимое статьи...</p>
        </article>
        
        <aside>
            <h3>Боковая панель</h3>
            <p>Дополнительная информация</p>
        </aside>
    </main>

    <footer>
        <p>&copy; 2023 Компания. Все права защищены.</p>
    </footer>
</body>
</html>
2. Какие новые семантические элементы появились в HTML5?
<header> - шапка сайта или раздела

<footer> - подвал сайта или раздела

<nav> - навигационное меню

<main> - основное содержимое страницы

<article> - независимая статья/запись

<section> - тематическая группа содержимого

<aside> - боковая панель, дополнительный контент

<figure> и <figcaption> - медиа-контент с подписью

<time> - для обозначения времени и даты

<mark> - выделенный текст

<progress> - индикатор прогресса

<meter> - скалярное значение в известном диапазоне

содержимого.

<samp>	Пример вывода программы

5. Перечислите и опишите типы позиционирования, которые вы знаете.
static	Стандартное позиционирование (по умолчанию)
relative	Относительное позиционирование от нормального положения
absolute	Абсолютное позиционирование относительно ближайшего позиционированного предка
fixed	Фиксированное позиционирование относительно окна браузера
sticky	"Липкое" позиционирование (переключается между relative и fixed)
6. Позиционирование внутри элемента.
.container {
    position: relative; 
    width: 400px;
    height: 300px;
}

.child {
    position: absolute;
    top: 50%;         
    left: 50%;        
    transform: translate(-50%, -50%); 
}
<div class="container">
    <div class="child">Центрированный элемент</div>
</div>

8. Перечислите и опишите глобальные атрибуты тега в HTML5.
id	Уникальный идентификатор элемента
class	Список классов для стилизации
style	Инлайновые CSS-стили
title	Всплывающая подсказка
lang	Язык содержимого элемента
dir	Направление текста (ltr, rtl)
data-*	Пользовательские данные
hidden	Скрывает элемент
tabindex	Порядок перехода по Tab
accesskey	Горячая клавиша для активации
contenteditable	Разрешает редактирование
draggable	Разрешает перетаскивание
spellcheck	Проверка орфографии
translate	Разрешает/запрещает перевод
<div id="main" class="container primary" 
     data-user-id="123" 
     title="Информация"
     contenteditable="true"
     tabindex="1">
    Редактируемый контент
</div>
